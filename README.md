# Star Database

## This script creates a database with the main properties of stars.

### The user can add, remove, and change the database easily.

### **Properties**
+ Name
+ Type
+ Temperature
+ Brightness
+ Mass
+ WaveLength
+ Radius 
+ Apparent Magnitude
+ Absolute Magnitude
+ Distance of Earth
